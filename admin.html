Règles de correspondance

Pour chaque formation, cochez les villes, domaines et niveaux qui déclenchent son affichage. Chaque modification est publiée instantanément sur GitHub.

Paramètres GitHub

Le back office publie les modifications directement dans votre dépôt GitHub via l'API. Un Personal Access Token avec la permission contents:write est requis.

Comment obtenir un token ?

  1. Allez sur github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Cliquez Generate new token
  3. Sélectionnez le repo esg-quiz
  4. Permission : Contents → Read and write
  5. Copiez le token généré ci-dessous
